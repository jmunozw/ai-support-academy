# AI Support Academy

Ruta práctica de aprendizaje y reconversión profesional para evolucionar desde Soporte IT hacia un perfil especializado en soporte, operación y automatización de sistemas de Inteligencia Artificial.

## Objetivo profesional

Convertirme en:

> AI Support & Operations Specialist

Aprovechando mi experiencia previa en soporte técnico, sistemas, redes y atención al usuario para aprender a desplegar, probar, monitorizar y resolver incidencias en aplicaciones basadas en IA.

## Punto de partida

Cuento con más de 10 años de experiencia relacionada con:

- HelpDesk y soporte técnico N1/N2
- Diagnóstico de incidencias
- Redes y telecomunicaciones
- Active Directory
- VMware ESXi y Veeam
- Soporte a usuarios
- HTML, CSS, JavaScript y React
- Python básico
- Git y GitHub

Esta academia está diseñada para ampliar ese perfil hacia la IA aplicada, sin orientarlo a ciencia de datos ni Machine Learning avanzado.

## Evolución profesional

1. AI Support Specialist
2. AI Automation Specialist
3. AI Operations / Customer Success técnico

## Método de aprendizaje

El proyecto sigue un proceso práctico:

1. Aprender
2. Practicar
3. Construir
4. Documentar
5. Validar
6. Mejorar
7. Empaquetar
8. Monetizar

Cada concepto debe producir alguna evidencia:

- Resumen técnico
- Ejercicio
- Laboratorio
- Ticket o incidencia
- Proyecto
- Documentación para portfolio
